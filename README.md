# Academic Terminal V2.0 - Student Scoreboard

A React-based student scoreboard application with a sci-fi/cyberpunk terminal aesthetic.

## Features

- **Add Students**: Register new students with their scores.
- **Dynamic Stats**: Real-time calculation of total students, passed students, and average score.
- **Interactive Rows**: Easily update a student's score inline with a dedicated "SAVE" button.
- **Visual Feedback**: Glowing green left borders for passing students (score >= 40) and red for failing students.
- **Cyberpunk Aesthetic**: Pure CSS styling with CRT flicker animations, glowing text, and monospace fonts.

## Tech Stack

- React.js (Vite)
- Pure CSS (No external libraries like Tailwind)
- Google Fonts (Share Tech Mono, Teko)

## Run Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
